# ML-Project-for-predicting-Student-s-Grades
A begginer project in data analysis, for an extraccuricular course at UPT

    Student Performance Prediction - Hybrid ML Model

Acest proiect utilizează Machine Learning (Regresie Liniară) pentru a prezice nota finală a studenților la examen, bazându-se pe factori academici, sociali și de stil de viață. Proiectul introduce conceptul de Indici Hibrizi pentru a îmbunătăți acuratețea predicției.

    Performanța Modelului

Modelul hibrid a obținut rezultate excelente, demonstrând o corelație puternică între variabilele alese și rezultatele finale:

    R2:   0.8568 (Explică ~85% din variația notelor)

    MAE:  0.98 puncte (Eroare medie sub 1 punct)

    MAPE: 1.46%

    Caracteristici Cheie

Ingineria Datelor (Feature Engineering): Crearea de indici compuși precum Social_Index (venit, educația părinților, influența colegilor) și Health_Index (somn, activitate fizică).

Analiza de Sensibilitate: Vizualizarea modului în care modificarea unei singure variabile (ex: orele de studiu) influențează direct nota prezisă.
